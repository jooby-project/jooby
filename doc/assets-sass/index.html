sass

sass-lang implementation from Sass (ruby) Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.

Make sure you already setup the assets module in your project!

dependency

<dependency>
  <groupId>org.jooby</groupId>
  <artifactId>jooby-assets-sass</artifactId>
  <version>1.0.0.CR7</version>
  <scope>provided</scope>
</dependency>

usage

assets {
 fileset {
   home: home.scss
 }
 pipeline {
   dev: [sass]
   dist: [sass]
 }
}

options

assets {
 ...
 sass {
   syntax: scss
   dev {
     sourceMap: inline
   }
 }
}
